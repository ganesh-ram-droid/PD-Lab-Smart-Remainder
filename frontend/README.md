# Context Reminder Frontend

React Native frontend foundation for **Dynamic Contextual Relevance Evaluation System for Adaptive Location-Based Reminders**.

## Stack

- Expo React Native
- React Navigation
- NativeWind
- Firebase Authentication
- Cloud Firestore client SDK
- Zustand
- Axios
- React Hook Form
- React Native Vector Icons

## Setup

```bash
cd frontend
npm install
copy .env.example .env
npm start
```

Use `10.0.2.2` instead of `localhost` for Android emulator API calls when the backend runs on your computer.

## Foundation Scope

This foundation includes splash flow, Firebase authentication screens, authenticated bottom tabs, stores, services, constants, validators, and reusable UI components.

Reminder features, analytics, maps, notifications, and context evaluation are intentionally not implemented yet.
